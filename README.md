- **Author(s):** 施威銘研究室

- **ISBN:** 9789863127628
- **Publisher:** 旗標科技股份有限公司

## Summary

這本書探討如何將語言模型融入各種應用中，提供了實用的技巧和方法。從詳解 OpenAI API 參數到設計記憶功能的聊天機器人，以及整合外部工具和建立串流輸出的 AI 聊天架構，讀者將學到如何有效地應用語言模型。這本書的目標是讓開發者能夠更好地利用語言模型來開發程式，為使用者提供更優質的使用體驗。
